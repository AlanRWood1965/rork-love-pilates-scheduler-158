# Love Pilates Booking App


## Features

- **Browse upcoming classes** — See a scrollable timetable of Pilates classes organised by date, showing class type, level, and time
- **Filter by class type** — Quickly filter between Mat, Reformer, Tower, and Wunda Chair Pilates
- **Filter by level** — Filter classes by Beginner, Intermediate, Transition, or Advanced
- **Class detail view** — Tap a class to see full details including description, level info, and a "Book Now" button that opens the Bookwhen booking page
- **Studio information** — An About screen with info about Love Pilates, instructor Karen, the Milngavie location, and links to the website
- **Pricing overview** — View membership and drop-in pricing options
- **Pull-to-refresh** — Refresh the timetable to see the latest schedule

## Design

- **Colour palette inspired by Love Pilates**: Warm rose/blush pink as the primary accent colour, paired with soft whites, light greys, and charcoal text — evoking a welcoming, premium wellness studio feel
- **Elegant typography** with clean sans-serif headings and body text
- **Class cards** with subtle shadows, rounded corners, and colour-coded left borders by class type (e.g. soft pink for Mat, warm coral for Reformer, dusty rose for Tower, mauve for Wunda Chair)
- **Bottom tab navigation** with smooth icons for Schedule, Classes, About, and Contact
- **Soft gradient header** with the Love Pilates branding
- **No harsh colours** — everything feels calm, inviting, and premium like a boutique studio app

## Screens

- **Schedule (Home)** — Date-grouped list of upcoming classes with horizontal date selector, class cards showing time, type, level, and spots indicator. Filter chips at the top for class type and level
- **Classes** — Overview of each class type (Mat, Reformer, Tower, Wunda Chair) with descriptions and what to expect at each level
- **About** — Studio info, Karen's bio, location in Milngavie (northwest Glasgow), contact details, and links to the Love Pilates website
- **Contact** — Enquiry form (name, email, optional phone, enquiry type picker, message) that composes an email to the studio via expo-mail-composer

## App Icon

- A stylish icon featuring a soft rose/blush pink background with an elegant white Pilates-inspired silhouette or abstract movement symbol, reflecting the Love Pilates brand's warm, welcoming aesthetic
